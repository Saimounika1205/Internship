# AI Chatbot using Hugging Face Transformers

## Description
This project is a conversational chatbot built using Microsoft's DialoGPT model from Hugging Face.

## Features
- Interactive chatbot
- Continuous conversation
- Exit condition
- Uses transformer-based NLP

## Technologies Used
- Python
- Hugging Face Transformers
- PyTorch

## How to Run
1. Install dependencies
2. Run notebook
3. Start chatting!
