# LangChain Assignment

## Objective
To demonstrate the use of LangChain with a free HuggingFace model.

## Technologies Used
- Python
- LangChain
- HuggingFace Transformers
- GPT-2 Model

## Features Implemented
1. Basic LLM Response
2. Prompt Chain
3. Memory Handling

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the file:
   python main.py

## Output
- AI explanation using LLM
- Machine Learning explanation using Chain
- Memory-based conversation

## Note
GPT-2 is a lightweight model, so outputs may not be highly accurate.
